Event Kafka Processor

A full-stack event-driven project built using:

Frontend: React  
Backend: Spring Boot  
Database: PostgreSQL  
Event Streaming: Apache Kafka  

This project demonstrates how events flow from producers to consumers through Kafka, 
and how the frontend interacts with backend APIs.

---

*Frontend
- React
- Axios
- React Router

*Backend
- Spring Boot
- Spring Kafka
- Spring Web
- PostgreSQL Driver
- Lombok

*Database
- PostgreSQL

- Apache Kafka (Producer + Consumer)
